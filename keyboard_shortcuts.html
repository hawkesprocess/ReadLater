Read Later Keyboard Shortcuts

The Read Later extension comes with a keyboard shortcut to make saving pages faster. If you're having trouble with the shortcut, this guide will help you set it up.

Default Shortcut

  • Alt+R - Save the current page (with tag selection)
How it works: When you press Alt+R, the extension popup will open, allowing you to add tags before saving the current page.

Fixing Keyboard Shortcuts

If your keyboard shortcut isn't working, you can manually set it in Chrome:

  1. Open Chrome and type chrome://extensions/shortcuts in the address bar
  2. Find "Read Later" in the list of extensions
  3. Click on the pencil icon next to the shortcut to modify it
  4. Press the key combination you want to use (e.g., Alt+R)
  5. Click outside the input box to save
Tip: If a shortcut is already in use by another extension or Chrome itself, you'll need to choose a different key combination.

Common Issues

Shortcut Not Working on Some Pages

Some websites may override keyboard shortcuts. Try a different combination if Alt+R doesn't work on specific sites.

Shortcut Conflict

Chrome will show you if there's a conflict with another extension or Chrome feature when you try to set a shortcut.

Extension Not Active

Make sure the extension is enabled. You can check by visiting chrome://extensions and making sure the toggle for Read Later is switched on.

Using Alternative Methods

If you're still having issues with the keyboard shortcut, you can also save pages using:

  • Clicking the Read Later extension icon in your toolbar
  • Right-clicking on a page and selecting "Save to Read Later"
